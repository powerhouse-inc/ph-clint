## When to use

Use this skill when you need to design or iterate on a Powerhouse document model. This includes defining state schemas, operations (mutations), and reducer logic.

## Expected inputs

- **prompt**: A description of the document type to model (e.g. "invoice document with line items and tax calculations")

## Example invocations

```
/document-modeling --prompt "Design an invoice document model with line items, tax rates, and payment status"
/document-modeling --prompt "Add a SET_DUE_DATE operation to the invoice model"
```

## Mode

This is a **conversational** skill — the agent will ask clarifying questions and iterate on the design with you.
