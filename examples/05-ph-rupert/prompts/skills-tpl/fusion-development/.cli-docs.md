## When to use

Use this skill when you need to build or modify pages and components in a Fusion (Next.js) application that uses document drives as the backend.

## Expected inputs

- **prompt**: A description of the feature or page to build (e.g. "create a dashboard page that shows all invoices")

## Example invocations

```
/fusion-development --prompt "Create a dashboard page listing all invoice documents with status filters"
/fusion-development --prompt "Add a document detail view with real-time collaboration indicators"
```

## Mode

This is a **conversational** skill — the agent will discuss approach and iterate on the implementation.
