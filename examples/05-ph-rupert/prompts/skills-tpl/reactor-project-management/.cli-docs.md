## When to use

Use this skill to initialize and manage Reactor Package projects. This includes creating new projects, starting Vetra services, and managing the development lifecycle.

## Expected inputs

- **prompt**: The management task to perform (e.g. "create a new reactor package" or "check service status")

## Example invocations

```
/reactor-project-management --prompt "Initialize a new Reactor Package project called 'my-docs'"
/reactor-project-management --prompt "Start the Vetra Studio server for the current project"
```

## Mode

This is a **one-shot** skill — the agent will complete the task without further interaction.
