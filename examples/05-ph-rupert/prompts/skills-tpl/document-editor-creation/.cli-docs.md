## When to use

Use this skill when you need to build or modify React editor components for Powerhouse document types. The editor provides the UI for viewing and editing document state.

## Expected inputs

- **prompt**: A description of what the editor should look like or do (e.g. "create a form-based editor for the invoice document model")

## Example invocations

```
/document-editor-creation --prompt "Create an editor for the invoice document with inline line-item editing"
/document-editor-creation --prompt "Add a status badge and payment tracking section to the invoice editor"
```

## Mode

This is a **conversational** skill — the agent will discuss layout options and iterate on the editor design.
