## When to use

Use this skill to initialize, configure, or run Fusion project instances. This includes setting up new Next.js projects with Powerhouse integration and managing dev server lifecycle.

## Expected inputs

- **prompt**: The management task to perform (e.g. "initialize a new Fusion project" or "configure the switchboard URL")

## Example invocations

```
/fusion-project-management --prompt "Initialize a new Fusion project called 'my-platform'"
/fusion-project-management --prompt "Start the Fusion dev server on port 3001"
```

## Mode

This is a **one-shot** skill — the agent will complete the task without further interaction.
