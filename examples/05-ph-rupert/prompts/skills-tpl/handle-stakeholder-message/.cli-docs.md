## When to use

Use this skill to triage incoming stakeholder messages. The agent will analyze the message, update relevant WBS (Work Breakdown Structure) documents, and draft a reply.

## Expected inputs

- **prompt**: The stakeholder message to process, or context about the message

## Example invocations

```
/handle-stakeholder-message --prompt "Client asks: Can we add multi-currency support to the invoice module?"
/handle-stakeholder-message --prompt "PM update: Sprint deadline moved to next Friday, re-prioritize backlog"
```

## Mode

This is a **one-shot** skill — the agent will triage, update documents, and draft a reply in one pass.
