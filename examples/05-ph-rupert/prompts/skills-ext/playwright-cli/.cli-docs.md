## When to use

Use this skill to interact with web pages through Playwright browser automation. Useful for testing UI components, taking screenshots, or extracting information from running applications.

## Expected inputs

- **prompt**: The browser automation task to perform (e.g. "navigate to the dashboard and take a screenshot")

## Example invocations

```
/playwright-cli --prompt "Navigate to http://localhost:3000 and take a screenshot of the dashboard"
/playwright-cli --prompt "Click the 'New Invoice' button and fill in the form fields"
```

## Mode

This is a **one-shot** skill — the agent executes the browser actions and reports results.
