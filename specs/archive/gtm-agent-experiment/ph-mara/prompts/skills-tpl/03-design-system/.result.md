# Expected Skill Outcome

`design-system/` contains a complete, self-contained visual identity and component library:
- `identity.md` — Visual identity rationale and choices
- `variables.css` — CSS custom properties (colors, fonts, spacing)
- `components.css` — All component styles using variables
- `preview.html` — All components rendered for visual review
- `assets/` — Self-contained fonts and icons

The site prototype (skill 04) will compose these components. No new visual patterns should be needed.
