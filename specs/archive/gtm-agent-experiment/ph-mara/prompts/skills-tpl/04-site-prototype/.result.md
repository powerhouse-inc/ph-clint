# Expected Skill Outcome

`site/` contains a complete, self-contained, serveable website:
- `index.html` — Homepage with all sections from approved messaging
- `css/` — Design system variables, components, and page layout
- `assets/` — Self-hosted fonts and icons
- `pages/` — Subpages (if multi-page structure)

The site is serveable with `npx serve site/` and contains no external dependencies. Run `/preview-server-start --directory site` for the managed preview.
